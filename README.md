# Number Analyzer (JavaScript)

This is a beginner JavaScript project that analyzes a single number.

## What the program does
- Counts how many digits the number has
- Calculates the sum of digits
- Counts even and odd digits
- Finds the maximum and minimum digit

## Rules
- Works with positive and negative numbers
- Handles the number 0 correctly
- Does not convert the number to a string

## Technologies
- JavaScript
- HTML

## JavaScript topics used
- while loop
- if / else
- modulo (%)
- parseInt

## How to run
1. Open `index.html` in a browser
2. Enter a number
3. Click analyze

## Example
Input:
5271
Output:
- Digits count: 4
- Sum of digits: 15
- Even digits: 1
- Odd digits: 3
- Max digit: 7
- Min digit: 1
